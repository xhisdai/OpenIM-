# OpenIM-
根据云旺IMSDK 不使用SDK自带的聊天页面 深度自定义聊天页面
###
如果编译demo发现列表为空 请用阿里官方自带demo和你登录的帐号先发送消息(或者下拉刷新)
###
需要拖入 官方Demo里面的文件夹 WXFrameworks 到本demo中
